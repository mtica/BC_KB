    What are system emails?

    System emails are triggered when a customer interacts with your site. For example, when a customer clicks the Lost Password link, the system sends an email to the customer.

    You can customize the content of system emails by editing their layout.

    List of available system emails

    • Invoice
    • Quote
    • Invoice Overdue
    • Order Status Changed
    • Lost Password
    • 3rd Party Workflow
    • Forum New Post Alert
    • Forum New Topic Alert
    • Double OptIn
    • Secure Zone Details
    • Affiliate Program Details
    • Gift Voucher
    • Drop Shipping
    • Workflow Notification
    • Webforms auto responders
    • Tell a Friend
    • CST layout
